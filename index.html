The Crystal Game

Rules:

For every round, there is a randomly generated "target" number. "Your" number always starts at 0. The four crystals are different buttons. Each colored crystal contains a hidden value. Every time you click a crystal, it will add that value to "your" number. Click the buttons in any combination you can to try to match "your" number with the target number. If "your" number matches the target number, you win! If "your" number ends up larger than the target number, you lose. When the game restarts, the target number changes, and each of the crystal values change as well. Have fun!
